# 信息标注规则

## Summary
- 张雪峰的公开表达呈现出非常强的“结论先行 + 现实后果导向 + 场景化问答”特征。
- 其标题与公开内容组织高度口语化，常直接点名群体、专业、结果，少做抽象铺垫。
- 他的表达辨识度主要来自“直接、具体、带判断、带劝阻/建议”的咨询风格，而不是书面修辞。

## Key Findings
- 高频结构之一是直接提问式：如“法学本科好就业吗？”“中低分数段女生有哪些选择？”标签：一手索引。
- 高频结构之二是否定纠偏式：如“不是所有学校的电气都是进电网的”。标签：一手索引。
- 高频结构之三是汇总答疑式：如“评论区问题精选，你关心的问题都在这里”。标签：一手索引。
- 从“文科就是服务业”相关视频标题可看出，他常用强判断、强压缩的表达把复杂现实打成一句好记的判断，容易传播，也容易引发争议。标签：二手索引。
- 表达节奏大概率是“先给一句结论，再拆适用边界”，而不是长铺垫后再出结论。标签：推断。
- 表达里的主语常是考生、家长、专业、学校、行业；说明他关注的不是抽象制度，而是“谁、多少分、去哪里、干什么”。标签：推断。
